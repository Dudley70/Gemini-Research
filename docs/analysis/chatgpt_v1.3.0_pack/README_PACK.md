# Custom GPT — Research Prompt Doc Pack (v1.2.0)
This pack lets a Custom GPT generate **adaptive research prompts** with deterministic gates.

## Files
- RESEARCH_FRAME_GUIDE.md — framework, determinism, Gate 0
- RESEARCH_TYPE_SCHEMA.yaml — types incl. **INTEGRATION_SCENARIOS**
- PAPER_PROMPT_BLUEPRINT.md — universal template
- STYLE_AND_SCORING_GUIDE.md — tone & rubrics
- SOCRATIC_QUESTION_LIBRARY.md — reusable questions
- QUALITY_GATES_LIBRARY.md — gates incl. **GATE_0_PREFLIGHT**
- PROJECT_CONTEXT.yaml — thresholds for PropTaxCalc
- INTEGRATION_CONFIG.yaml — load order guide
- CUSTOM_GPT_INSTRUCTIONS.txt — copy‑paste for the Instructions box
- COMPLIANCE_REPORT_SCHEMA.json — schema to validate final report
- SCHEMA_TEMPLATES/ — CSV schemas for IO Contracts & Matrices
- REQUEST_SPEC_TEMPLATE.yaml — form to request a new paper
